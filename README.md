# iForgot
A simple todo-list app using Mongo, Express, and Node

## Dependencies
* [NodeJS](https://nodejs.org/en/download/)
* npm

## Setup
1. `git clone https://github.com/edwardsCam/iForgot.git`
2. `cd iForgot`
3. `npm install`

## Run
1. `npm start`
2. Open up `http://localhost:3000`

## Test
1. `npm test`
